    • Minecraft

    Want to play on a Minecraft server where everyone treats everyone with respect, where everyone helps each other and run a simple survival server without too many mods? Look no further! Come join us! The server is ran on Java, but we have a plugin that lets Bedrock players join too!

    (you need to wait for an admin or the owner to join in order to grant you permissions to build and similar)

    Join the server

    Please be aware that you need to follow these rules in order to be permitted on the server:


    No one under 13 years old


    No griefing or stealing


    Be kind to others


    Don't hurt the villagers, iron golems or damage the houses at spawn


    No nuking the server


    Don't use hacked clients or x-ray tools

    Server Name: aridanMC

    Server Address: mc.aridan.net

    Port (if on Bedrock): 25609

    Server Resource Packs: Enabled
